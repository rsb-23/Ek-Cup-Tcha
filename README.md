Captcha solver extension

## Installation Steps

1. Download the latest release (.zip) from the [Releases]
2. Unzip the file
3. In Chrome, go to chrome://extensions/
4. Enable "Developer mode"
5. Click "Load unpacked"
6. Select the unzipped folder
